# DD监控室 v3.50魔改版 Release Notes

发布日期：2026-03-07

## 主要更新

- 完成弹幕 v3 改造：滚动 / 顶部 / 底部弹幕链路统一，默认开启。
- 播放器交互优化：
  - 修复切回软件时可能误触拖拽。
  - 未播放窗口改为灰底占位，避免黑屏观感。
  - 底部控制栏点击保活，避免点弹幕后立即消失。
- 列表与数据稳定性：
  - 房号键一致性修复，新增/删除/置顶刷新链路收口。
  - 封面刷新策略优化，减少不刷新与重复下载。
- 网络与日志：
  - 流地址合法性过滤，避免 `Invalid URL ''`。
  - unknown cmd 日志限流，降低刷屏噪声。
  - 热门列表 `-352` 场景可自动 fallback。

## 升级说明

- Windows 打包默认版本号已切换为 `3.50`。
- 旧配置可继续使用；建议升级后执行一次手动全局刷新。
