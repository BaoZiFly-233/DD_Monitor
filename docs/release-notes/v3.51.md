# DDMonitor v3.51 (2026-05-05)

## 修复

### 配置系统
- **房间列表持久化**：修复 `roomid` 配置不回写导致关闭重开后房间列表丢失
- **备份轮转**：重写备份逻辑，从同步覆盖改为真正的轮转备份（config → 备份1 → 备份2 → 备份3），确保至少保留一个历史版本
- 修复 `credentialRefresh` 无凭据时静默失败（现在正确 emit `failed` 信号）
- 修复 `closeEvent` 未调用 `accept()` 可能导致窗口关闭异常
- 修复 `changeLayout` 中 `itemAt(0)` 可能为 `None` 导致崩溃

### 弹幕系统
- **实现缺失的 `GlobalDanmuOption`**：全局弹幕设置面板（之前代码引用了不存在的类）
- **实现缺失的 `DanmakuEvent`**：标准化弹幕事件数据类（之前 import 不存在的类）
- **新增 `RollingOptionWidget`**：滚动弹幕设置面板
- 修复 hex 颜色计算错误（`browser_opacity / 101 * 256` → 正确的 alpha 通道计算）
- 修复 `DanmakuHandler` 未调用 `super().__init__()`

### 线程安全
- 修复 `RecordThread` 跨线程数据竞争：`recordToken` / `downloadToken` 加锁保护
- 修复 `RecordThread` 文件写入异常时句柄泄漏（`open()` → `try/finally close()`）
- 修复 vtb.csv 两处 `open/close` 不安全用法（改为 `with open()`）
- 修复 `refreshHacoList` 双击导致 `RuntimeError`（重复 disconnect 已保护）

### 其他修复
- 修复 `pay.py` `bossList` 为空时 `NameError`
- 修复 `Slider` 宽度硬编码 100px（改为按实际宽度比例映射）
- 修复 `load_mpv_module` 导入失败后永久不重试
- 修复 `loginBrowser.setCredential` 方法不存在（改为 `setSessionData`）
- 修复 `checkUpdate` 删除 `latest` signal 后残留引用
- 修复 `deleteAll` 中 `CoverLabel` 从不释放导致内存泄漏
- 修复 `ReportException.py` `\u` 转义错误 + `thraeding` 拼写错误
- 修复 DNS 异常日志不输出实际错误信息
- 移除 `Connection: close` 头（恢复 HTTP 连接池复用）
- 修复 `user-agent` 默认回退 header

## 重构

- **remote.py**：`asyncio.Event()` 替代 busy-wait `asyncio.sleep(0.5)`，优雅停线
- **remote.py**：新增线性退避重连策略（1s → 2s → 封顶 30s），避免 32 窗口同时重连风暴
- `config_manager.py`：备份轮转从同步覆盖改为真正轮转
- `ConfigManager` 添加 `parent` 参数，防止 use-after-free

## 清理

- 删除 `webBrowser.py`、`sample.py`、`test.py`（死文件）
- 删除 `MpvDanmakuRenderer`（278行 ASS 字幕死代码，已被 `DanmakuRenderer` 替代）
- 删除 `latestRemainder` 类 + `latest` signal
- 删除 `plugins/`、`plugins_lite/`（VLC 残留 ~77MB）
- 删除 `docs/plans/`（AI 开发文档）
- 去重 `DISPLAY_RATIOS` 常量
- 添加 `blivedm/` vendored 版本注释
