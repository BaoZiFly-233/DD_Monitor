# DDMonitor v3.51

> 自 v2.26 以来的首个大版本更新。底层从 VLC 全面迁移至 MPV + OpenGL，弹幕系统完全重写，UI 交互大幅升级。

---

## 🎯 底层架构

### MPV 播放器内核（替代 VLC）
- VLC 插件依赖完全移除，零外部运行时
- MPV `libmpv-2.dll` 嵌入式播放，支持硬件解码
- OpenGL 渲染路径（`MpvGLWidget`），GPU 加速视频 + 弹幕叠加层
- Windows 下智能禁用硬解以规避 libmpv + ANGLE 花屏问题
- 五档画质切换：原画 / 蓝光 / 超清 / 流畅 / 仅音频
- 单窗口音量增强 0.5x ~ 4.0x

### 弹幕系统重写（v3 魔改版核心）
- **全新滚动弹幕渲染器**：MPV `osd-overlay` + QPainter 叠加，无临时 ASS 文件、无 `sub-reload` 闪烁
- **弹幕布局引擎**：独立模块 `danmaku_layout.py` — 滚动/顶部/底部三轨道管理
- **精灵缓存系统**：`DanmakuImageCache`，512 条目 LRU 缓存，大幅降低 CPU 占用
- **全局弹幕设置面板**：`GlobalDanmuOption` — 弹幕窗 + 滚动弹幕统一配置界面
- 弹幕机支持透明度/字体/比例独立调节
- 同传过滤、礼物/进入信息分类显示

### 扫码登录
- 全新 `QRLoginWidget` 模块，替代旧 QWebEngine 方案
- Session 自动验证 + Token 刷新
- 关注列表自动获取，卡片面板持久化

---

## 🛠️ 可靠性

### 配置系统
- `ConfigManager` 独立模块：加载 → 迁移 → 去抖动保存
- **备份轮转**：config → 备份1 → 备份2 → 备份3，每次保存保留上一轮完整配置
- 配置导入/导出（JSON 格式），预设管理
- 启动自动从损坏配置中恢复

### 线程安全
- `RecordThread` 跨线程状态加锁保护
- `remote.py` 弹幕线程：`asyncio.Event` 替代 busy-wait，优雅启停
- HTTP 连接池复用（`requests.Session` + `HTTPAdapter`）
- 线性退避重连策略（1s → 2s → 封顶 30s），防止多窗口重连风暴

### Bug 修复（28+ 项）
- 房间列表关闭重开后不再丢失
- hex 颜色计算修复
- 多处文件句柄泄漏修复
- Slider 宽度按比例映射（不再硬编码 100px）
- 内存泄漏修复（CoverLabel 永不释放）
- 多个崩溃级引用缺失修复

## 📁 项目结构

```
DD监控室.py              # 主窗口入口
config_manager.py         # 配置管理（加载/保存/迁移/备份）
VideoWidget_mpv.py        # MPV 播放器窗口
danmu.py                  # 弹幕系统（弹幕机 UI、全局设置）
danmaku_renderer.py       # 滚动弹幕渲染器（OpenGL 叠加层）
danmaku_layout.py         # 弹幕布局引擎
LiverSelect.py            # 主播卡片面板
login.py                  # 扫码登录
remote.py                 # 弹幕 WebSocket 接收线程
http_utils.py             # 共享 HTTP 连接池
mpv_gl_widget.py          # MPV OpenGL 渲染控件
blivedm/                  # B站弹幕协议库（vendored）
```
